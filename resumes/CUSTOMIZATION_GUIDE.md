# Resume Customization Guide

Quick guide for customizing your resume for each job application.

## Quick Customization Process

### Step 1: Analyze the Job Description (5 minutes)

**Identify:**
- [ ] Required skills (technical and soft skills)
- [ ] Preferred qualifications
- [ ] Key responsibilities mentioned
- [ ] Industry-specific keywords
- [ ] Company culture/values mentioned

**Top 5 Requirements:**
1. [List requirement 1]
2. [List requirement 2]
3. [List requirement 3]
4. [List requirement 4]
5. [List requirement 5]

### Step 2: Match Your Experience (10 minutes)

**For each requirement, identify:**
- Which of your experiences demonstrate this skill
- Specific achievements or examples
- Quantifiable results if possible

### Step 3: Customize Resume Sections (15 minutes)

#### Professional Summary
- [ ] Match first 2-3 sentences to job requirements
- [ ] Include keywords from job description
- [ ] Emphasize most relevant experience
- [ ] Keep it concise (3-4 sentences)

#### Skills Section
- [ ] Reorder skills to match job priorities
- [ ] Place most relevant skills first
- [ ] Use exact terminology from job description
- [ ] Remove or de-emphasize irrelevant skills

#### Experience Section
- [ ] Reorder bullet points within each role
- [ ] Emphasize most relevant responsibilities
- [ ] Add keywords naturally
- [ ] Quantify achievements where possible
- [ ] Match job title language

#### Projects Section (if including)
- [ ] Emphasize most relevant projects
- [ ] Highlight technologies they use
- [ ] Show results/impact

### Step 4: Final Review (5 minutes)

- [ ] Read job description one more time
- [ ] Check that all key requirements are addressed
- [ ] Verify keywords are included naturally
- [ ] Ensure formatting is clean and ATS-friendly
- [ ] Spell check and grammar check

## ATS (Applicant Tracking System) Tips

1. **Use Standard Section Headings**
   - Experience (not "Work History")
   - Skills (not "Technical Skills")
   - Education (not "Academic Background")

2. **Include Keywords Naturally**
   - Don't stuff keywords
   - Use them in context
   - Match job description terminology

3. **Formatting**
   - Use standard fonts (Arial, Calibri, Times New Roman)
   - Keep formatting simple
   - Avoid tables, graphics, or complex layouts
   - Use standard date formats (Month YYYY - Month YYYY)

4. **File Format**
   - Save as PDF
   - Use descriptive filename
   - Keep file size reasonable (<2MB)

5. **Contact Information**
   - Include at top of resume
   - Use standard format
   - Include email, phone, LinkedIn, website

## Tracking Your Applications

**Keep a spreadsheet with:**
- Company name
- Position title
- Date applied
- Resume version used
- Response received
- Notes/feedback

**Filename convention:**
`[Company]_[Position]_[Date].pdf`

Example: `AcmeCorp_SoftwareEngineer_2025-11-03.pdf`

## Pro Tips

1. **Save Originals:** Always keep your base resume as a backup
2. **Version Control:** Use descriptive filenames so you know which version you sent
3. **Track Responses:** Note which customizations get responses
4. **Learn Patterns:** Over time, you'll see which customizations work best
5. **Be Genuine:** Customize authentically - don't oversell skills you don't have
6. **Keep It Real:** Customize emphasis, not facts - stay truthful
