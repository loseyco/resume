# Example Job Application Folder

This is an example structure showing how to organize a job application.

## Files in This Folder

- **job_posting.md** - The original job posting you want to apply for
- **analysis.md** - AI analysis of the job requirements and how your experience matches
- **resume.md** - Your customized resume for this specific role
- **resume.html** - HTML version ready for PDF generation
- **notes.md** - Discussion notes, revisions, and any questions

## How to Use

1. Copy this folder structure for each new job application
2. Rename the folder to: `CompanyName_PositionTitle`
3. Fill in `job_posting.md` with the job details
4. Ask me to analyze it and create your customized resume
