# Application Tracker

Track your job applications and which resume version you used for each.

## Quick Reference

| Date | Company | Position | Resume Version | Status | Notes |
|------|---------|----------|----------------|--------|-------|
| YYYY-MM-DD | Company Name | Position Title | Filename | Applied/Interview/Rejected/Offer | Notes |
| | | | | | |
| | | | | | |
| | | | | | |

## Status Legend

- **Applied** - Resume submitted
- **Acknowledged** - Received confirmation
- **Screening** - Phone/initial screening
- **Interview** - Interview scheduled/completed
- **Follow-up** - Waiting for response
- **Rejected** - Not selected
- **Offer** - Received offer
- **Accepted** - Accepted position

## Notes Template

**Company:** [Name]  
**Position:** [Title]  
**Date Applied:** [Date]  
**Resume Used:** [Filename]  
**Application Method:** [LinkedIn/Email/Company Portal/Referral]  
**Status:** [Current Status]  
**Follow-up Date:** [Date]  
**Key Requirements:** [List top 3-5 requirements]  
**Customizations Made:** [What you emphasized/changed]  
**Response/Feedback:** [Any feedback received]  
**Next Steps:** [Follow-up actions needed]
