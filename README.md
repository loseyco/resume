## Deployment

### Option 1: Netlify (Recommended - Free)

1. Sign up at [netlify.com](https://netlify.com)
2. Drag and drop the project folder to Netlify
3. Add your custom domain (resume.losey.co)
4. Done! Your site is live

### Option 2: Vercel (Free)

1. Sign up at [vercel.com](https://vercel.com)
2. Connect your GitHub repository
3. Deploy with one click
4. Configure custom domain

### Option 3: GitHub Pages (Free & Easy!)

1. Push all files to your repository: [github.com/loseyco/resume](https://github.com/loseyco/resume)
2. Go to **Settings ‚Üí Pages** in your GitHub repository
3. Under "Source", select **"Deploy from a branch"**
4. Choose **"main"** branch and **"/ (root)"** folder
5. Click **Save**
6. Your site will be live at `loseyco.github.io/resume`
7. Optionally, add a custom domain (resume.losey.co) in the same Settings page

### Option 4: Traditional Web Hosting

1. Upload all files via FTP
2. Point your domain to the hosting
3. Configure DNS settings
4. Site is live!

## Customization

### Update Personal Information

1. **HTML (`index.html`):**
   - Replace all instances of "PJ Losey" with your name
   - Update email addresses
   - Modify social links
   - Customize project descriptions
   - Update experience section

2. **Styling (`styles.css`):**
   - Change color scheme in `:root` variables
   - Adjust fonts
   - Modify spacing and sizing

3. **JavaScript (`script.js`):**
   - Update typewriter phrases
   - Modify animations
   - Add custom interactions

### Adding Your Photo

Replace the placeholder in the About section:
1. Add your image to an `images` folder
2. Update the `about-image` section in `index.html`
3. Replace placeholder with `<img src="images/your-photo.jpg" alt="Your Name">`

### Adding Resume PDF

1. Export your resume as PDF
2. Save as `resume.pdf` in the root directory
3. The download button will automatically work
