# Documentation

This folder contains all documentation and guides for the resume website project.

## Files

- **CHECKLIST.md** - Pre-deployment checklist
- **DEPLOYMENT.md** - Deployment instructions
- **GITHUB_SETUP.md** - GitHub repository setup guide
- **PHOTO_DOWNLOAD_INSTRUCTIONS.md** - Instructions for downloading and organizing photos
- **STATS_SETUP.md** - Page view tracking setup guide
- **PERSONAL_MESSAGES.md** - Message templates for outreach
- **SOCIAL_MEDIA_POSTS.md** - Social media post templates
- **RESUME.md** - Main resume in Markdown format
- **RESUME_HTML.html** - HTML version of resume for PDF generation
